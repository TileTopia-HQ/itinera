Features

🧮 Routing Algorithms

  • Dijkstra shortest-path
  • A* with haversine heuristic
  • Contraction Hierarchies (sub-ms)
  • Bidirectional CH search
  • Shortcut unpacking for full paths

🗺️ OSM Import

  • XML (.osm) parser
  • PBF (.osm.pbf) via osmpbf crate
  • Highway tag classification
  • One-way detection
  • Turn restriction parsing
  • Max speed from tags

📊 Graph Engine

  • Compressed Sparse Row (CSR)
  • Reverse CSR for bidirectional search
  • R-tree spatial index (rstar)
  • Bincode serialization
  • Incremental shortcut insertion

🚗 Routing Profiles

  • Car (motorway-optimized)
  • Bicycle (avoids motorways)
  • Pedestrian (walking speed)
  • Truck (weight-restricted)
  • Custom profiles via speed tables

🧭 Navigation

  • Turn-by-turn instructions
  • Bearing-based maneuver detection
  • Road name annotations
  • Depart / Arrive markers
  • Slight / Sharp / U-turn classification

🌐 HTTP API

  • Axum-based REST server
  • Route / Nearest / Isochrone / Health
  • Profile selection via query param
  • Algorithm selection (dijkstra/astar/ch)
  • CORS support for browsers

Quick Start

# Build from source
git clone https://github.com/TileTopia-HQ/itinera.git
cd itinera && cargo build --release

# Import OSM data
itinera import --input region.osm.pbf --output graph.bin --profile car

# Pre-build Contraction Hierarchies
itinera preprocess --graph graph.bin --output ch.bin

# Start the routing server
itinera serve --bind 0.0.0.0:5000 --graph graph.bin --ch ch.bin

# Query a route
curl "http://localhost:5000/route?from=48.8566,2.3522&to=48.8738,2.2950&profile=car"
